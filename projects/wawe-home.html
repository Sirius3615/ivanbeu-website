Wawe Home

Published on 2023 • Smart Home

Project Overview

Wawe Home is a revolutionary smart home startup focused on simplifying home automation through modular, easy-to-use IoT devices.

Technical Stack

  • Flutter for cross-platform mobile app development
  • Appwrite backend for secure, scalable cloud services
  • ESP32 for IoT device functionality
  • Custom PCB design and circuit development

Key Features

  • Modular IoT device ecosystem
  • Cross-platform mobile application
  • Real-time device control and monitoring
  • Secure cloud infrastructure
  • Custom hardware solutions
